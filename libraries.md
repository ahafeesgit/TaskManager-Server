## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deployment

When you're ready to deploy your NestJS application to production, there are some key steps you can take to ensure it runs as efficiently as possible. Check out the [deployment documentation](https://docs.nestjs.com/deployment) for more information.

If you are looking for a cloud-based platform to deploy your NestJS application, check out [Mau](https://mau.nestjs.com), our official platform for deploying NestJS applications on AWS. Mau makes deployment straightforward and fast, requiring just a few simple steps:

```bash
$ npm install -g @nestjs/mau
$ mau deploy
```

With Mau, you can deploy your application in just a few clicks, allowing you to focus on building features rather than managing infrastructure.

# Task Logger

Task Logger is a NestJS-based server for logging and managing tasks. This project is based on a robust template and includes a curated set of libraries for building secure and scalable applications. Below is a list of included libraries and their purposes:

## Core Libraries

- **@nestjs/core, @nestjs/common, @nestjs/platform-express**: Core NestJS framework modules.
- **@nestjs/config**: Environment variable and configuration management.
- **@nestjs/swagger**: API documentation with Swagger/OpenAPI.
- **@nestjs/throttler**: Rate limiting for APIs.
- **@nestjs/serve-static**: Serve static files (images, docs, etc.).
- **@nestjs/schedule**: Cron jobs and scheduled tasks.
- **@nestjs/terminus**: Health checks and monitoring endpoints.
- **@nestjs/cache-manager**: Caching support (in-memory, Redis, etc.).
- **@nestjs/platform-i18n**: Internationalization (i18n) support.
- **@nestjs/graphql**: GraphQL API support.

## Security & Middleware

- **helmet**: Sets secure HTTP headers.
- **compression**: Gzip compression for responses.
- **cookie-parser**: Parses cookies from requests.
- **express-rate-limit**: Rate limiting middleware.
- **cors**: Cross-Origin Resource Sharing support.
- **multer**: File upload handling middleware.

## Utilities

- **class-validator, class-transformer**: DTO validation and transformation.
- **uuid**: Generate unique IDs.
- **dayjs**: Lightweight date/time manipulation.
- **rxjs**: Reactive programming utilities (used by NestJS).

## Logging & Monitoring

- **pino, pino-pretty**: Fast, structured logging.
- **@sentry/node**: Error monitoring and reporting.
- **opentelemetry-api, opentelemetry-sdk-node**: Distributed tracing and observability.

## ORM/Database

- **prisma, @prisma/client**: Type-safe database ORM for PostgreSQL and other databases.
